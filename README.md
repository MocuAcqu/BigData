  ## 功能說明
  這是一個可以使用在Gmail信件的Chrome擴充工具:釣魚信件偵測器，他主要分成兩個功能:
  1. 批次分析功能
     它可在Gmail信箱中，去以"頁"為單位分析每則信件是否包含異常標題、寄信人，並將異常信件顯示在popup視窗中，說明異常原因，警惕使用者需要注意該信件。
     
  2. 單封信分析功能
     當使用者點入任意一封信件中，可以使用此功能去分析該封信記中的標題、寄信人、內容、附件，是否包含異常關鍵字、釣魚網址。
